Esercizio

Milestone 1
Creare una pagina che visualizzi una lista di articoli, mostrandone solo il titolo.

Milestone 2
Aggiungiamo in pagina un semplice form con un campo input in cui inserire il titolo di un nuovo articolo del blog. Al submit del form, mostrare la lista degli articoli aggiornati.
 
NOTATE BENE
Nell'esercizo di oggi si lavorarà solo con un campo, quindi un array di stringhe va più che bene per gestire i titoli. Nei prossimi giorni lavoreremo con dati più complessi (array di oggetti), però per oggi manteniamoci sul semplice.

BONUS
 
Aggiungere la possibilità di cancellare ciascun articolo utilizzando un'icona.
Implementare la funzionalità di modifica del titolo di un post.
Impostare il lavoro su più componenti.
